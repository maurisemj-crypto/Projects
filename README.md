My first project is an application of the Monte Carlo Model to approximate Pi. This method is highly innefficient due to the convergence rate of 1/(sqrt N), however, in order to demonstrate the model,
I have used this simple concept.
My second project uses RSA, however, due to the limitations of my laptops computational power, I was unable to use secrets.random, and I was unable to viably use 2048 bit primes due to the inefficiency of the 
Miller Rabin test. While the probability of failure is low, the time taken to find primes is still infeasible. Hence, I attempted to streamline this encryption by ECC Encryption,
which failed due to a paucity of time.
